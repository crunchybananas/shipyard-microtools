## why this shape

- **Closes the filed prose exactly.** 116's
  filed idea: *"Old folk still call them
  ${shape}."* Shipped to spec.
- **Reads as the NAME persisting**, not the
  ACT of naming being repeated. Default echo
  frames quote the source verbatim:
  - *"An elder is heard telling a child:
    'The first autumn stars stand clear above
    the eastern ridge. ${f}, looking up, names
    them the Hare. The realm takes the name.'"*
  - That's a 50+ word echo of an 80-word
    source. Reads as "rehashing the original
    moment."
  - 194's frame: *"Old folk still call them
    the Hare."* — 8 words. Reads as "this is
    the name now, generations later."
- **Detection by unique phrase.** "first autumn
  stars stand clear above the eastern ridge"
  is unique to the 116 entry's text generator.
  Both founder-named and elder-named variants
  start with that phrase.
- **Shape extraction by regex.** Matches both
  variants:
  - `${f}, looking up, names them the Hare.
    The realm takes the name.` →
    `names them` → `the Hare`
  - `The elders name the pattern the Hare.
    The realm takes the name.` →
    `name the pattern` → `the Hare`
  - Fallback to `them` if the regex misses
    (defensive — shouldn't happen with current
    text but safer than throwing).
- **Sibling to existing constellation arc.**
  - 116 names it (year-1 autumn).
  - 151 renders it (visual).
  - 159 shows a star moving through (silent
    visual moment).
  - 190 closes it (forgotten).
  - **194 echoes it (memory persistence in
    chronicle)** — fits between naming and
    forgetting as the "still called" stage.
  Now the constellation arc has 5 touchpoints
  across narrative + visual + memory.
- **No new echo frame in the table.** The
  generic 4-frame pool stays at 4 (one for
  each elder/traveler/bard/old-folk register).
  194 adds a CONDITIONAL fast-path before the
  generic dispatch. Could refactor later to a
  source-tag-specific lookup table if more
  source-specific echoes accrete.
