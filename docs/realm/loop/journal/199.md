## why this shape

- **Direct response to 196 filing.** "first
  cold morning (year-1 autumn day 15+, before
  constellation_named fires)." Ships the
  candidate exactly per filing — same day-
  window, placed before constellation in table
  order so morning-beat fires first.
- **Morning-sibling to 196 evening-beat.**
  first_long_evening = summer day ≤ 14
  (anti-anxious, work-done, fire-burning-low).
  first_cold_morning = autumn day ≥ 15 (turning,
  observational, "goes on"). Two early-game
  beats now bracket year-1's mid-game with
  morning/evening register-pair.
- **"Goes on" closes hopeful-not-anxious.**
  The 196 anti-anxious mood wasn't a one-off
  experiment — early-game beats deserve
  continuity-mood not alarm-mood. Cold-morning
  could have read alarm ("winter is coming")
  but the closing "goes on" lands as resilient
  routine. Realm knows seasons turn.
- **Observational-elder register.** Joins the
  148/152/166/174/184/190/193/196 cluster.
  Misc-tag is now 10 entries (largest cluster
  after milestone). The cluster has internal
  consistency (declarative present-tense, no
  founder reference, "the realm" subject).
- **Static prose, no kingdom-hash.** Cold
  mornings are universal — first frost varies
  by climate but every northern realm
  experiences this. Hash-variants would
  introduce false specificity. Matches 196's
  same choice for the same reason.
- **Same gate as 116 by design.** Same-day
  beat-pairing creates a literary moment: the
  realm experiences the same day at two
  registers. Not a bug — a feature. The
  for-of NARRATIVE_BEATS loop fires both on
  day 15 in chronicle order.
- **Year-1 only via flag-once-fire.**
  Following 196's pattern: no explicit year
  guard. Once-fire semantics ensure it fires
  for the first realm-autumn the player
  reaches. For brand-new saves that's year-1.
- **Anti-pattern check.** Does this duplicate
  174 first_frost_marked? No — 174 is year-2+
  autumn day ≥ 45 (5-day window), explicitly
  about the realm RECOGNIZING the change after
  one full cycle. 199 is year-1 first
  autumn-cold morning, BEFORE the realm has a
  reference frame. Different beat, different
  year, different gate.
