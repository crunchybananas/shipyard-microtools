## why this shape

- **Cross-axis triangle for barracks complete.**
  Canvas pre-loop + 182 SVG + 183 3D. 7 of 11
  buildings now expressed across all three
  layers (counting castle/church/house's
  implicit pre-loop-plus-SVG triangles). The
  cross-axis pattern is now a clear majority
  state.
- **All-pushBox primitives.** No new mesh
  primitive needed — barracks is rectangular
  enough that pushBox + per-crenellation pushBox
  expresses the shape. `pushCylinder` (163) is
  still the only post-loop mesh primitive
  added; barracks doesn't need it.
- **Reusable form template.** The 5-crenellation
  for-loop pattern could apply to tower's
  parapet (already pre-loop with a single
  parapet box), castle's central keep parapet,
  or any future fortified building. Filed.
- **Banner as static red box.** The flag in 3D
  doesn't sway. Same restraint as 167 windmill
  (static blades). Animation deferred to a
  future shader-side tick OR a dedicated 3D
  animation tick.
- **Defensive-industrial palette consistent.**
  Stone color [0.46, 0.51, 0.58] cool grey-blue,
  matching 182 SVG's `#4a5260` shadow tone scaled
  down. Banner red [0.82, 0.20, 0.18] tonally
  similar to SVG's `#a82820`.
