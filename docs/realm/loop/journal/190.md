## why this shape

- **Closes the constellation arc as a 2-beat
  cycle** (vs stone's 4-beat). 116 named the
  constellation; 190 says no one names it
  anymore. The OBJECT-ARC CLOSURE TEMPLATE (188
  invariant) doesn't require 4 beats first —
  only that the object has earned its closure.
  Constellation has 1 narrative anchor (116) +
  visual support (151 SVG asterism + 159
  shooting star) — sufficient earned-presence
  for a closure.
- **Generational-gap framing matches 184/147.**
  "The youngest...other names, or by no name
  at all" parallels stone's "youngest...never
  heard the story" and great-storm's "children
  born after do not believe it." Three beats
  now use the same generational-gap idiom for
  three different focal axes (object: stone;
  weather memory: storm; sky pattern:
  constellation).
- **Tag: milestone matches 116's anchor.** Per
  188 discipline rule: "tag inherits from
  focal-object's existing tag." 116 used
  milestone; 190 uses milestone.
- **Static prose, no kingdom-hashed variants.**
  Per 188 rule: "static prose preferred for
  closure (forgetting is universal)." Where
  116's text picks a kingdom-hashed shape
  name, 190's text doesn't NAME the
  constellation — that's the point. The
  shape that 116 picked goes generic again.
- **Year-3 + day ≥ 85 gate.** Year3 starts
  day 57; `day >= 85` pushes into deep year 3
  / year 4 (which 092/093 deliberately skipped
  as a flag). Realm needs ~30 days of
  "having a constellation" before the
  forgetting fires.
- **citizensBorn ≥ 10 turnover.** Matches 184
  stone-forgotten's gate. 188 invariant rule
  honored.
- **Different cycle shape from stone.** Stone:
  discovery → use → weathering → forgetting
  (4 beats). Constellation: named →
  forgotten (2 beats). Both close cleanly.
  Template tolerates either.
