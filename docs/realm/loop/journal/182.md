## what barracks.svg expresses that canvas struggles with

- **Weapons rack with shield.** Canvas barracks
  has only the training dummy as exterior
  detail. SVG adds a full weapons rack
  (3 spears + shield with center boss + red-
  cross emblem) — applies the "tools propped/
  hanging" idiom from 181's milestone review
  (sibling to blacksmith's tongs/hammer rack).
- **Cruciform arrow slits.** Canvas has plain
  rect slits with simple horizontal cross-bar.
  SVG ships proper cruciform shape matching
  tower 170 — historically authentic medieval
  arrow loop.
- **Painted target marks on dummy head.**
  Canvas dummy is a plain blob ellipse. SVG
  adds 2 dark dots + horizontal mark
  representing the target circles painted on
  practice mannequins.
- **Worn/struck spots on dummy body.** Canvas
  dummy is uniform color. SVG adds 3 small
  dark spots implying "this dummy has been
  beaten on for a while."
- **Shield boss and red-cross emblem.** Canvas
  has no shield; SVG adds a full round shield
  with metal center boss + heraldry-style cross.
