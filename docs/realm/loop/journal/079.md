# 079 — surprise (offering at the stone)

**Challenge:** Surprise — un-filed, not in ideas.md.
Last surprise was 072 (7 ticks ago) — at Evolution-
mandate threshold. 079 resets.

**What it does:** once per realm, on a dawn where
both the 056 stone has been placed AND the 071
happiness peak flag is active, ~15% chance to fire
a chronicle beat describing an anonymous offering
left at the stone. Cross-system compound — the
beat CAN'T fire unless the realm has progressed far
enough that multiple earlier systems have landed.

## the build

`docs/realm/js/story.js` — new `checkOfferingBeat()`
function + 6-item pool, wired into `checkStoryBeats`
after 072's founders check.

```js
const _OFFERING_ITEMS = [
  'a basket of new bread',
  'a rush of summer wildflowers',
  'a string of carved beads',
  'a small blue stone the rivers have not seen',
  'a child\'s drawing of the kingdom\'s gate',
  'a folded linen cloth the wind does not take',
];

export function checkOfferingBeat() {
  if (hasFlag('offering_made')) return;
  if (!hasFlag('stone_found')) return;
  if (!hasFlag('happyPeakActive')) return;
  if (G.dayPhase > 120) return;
  if (G.day < 30) return;
  const kname = G.kingdomName || 'Realm';
  const roll = _dreamHash(`${kname}_offering_${G.day}`) % 100;
  if (roll >= 15) return;
  setFlag('offering_made');
  const idx = _dreamHash(`${kname}_offering_item`) % _OFFERING_ITEMS.length;
  chronicle(
    `At dawn, ${_OFFERING_ITEMS[idx]} was found at the standing stone. No one admits to placing it there.`,
    'stone'
  );
}
```

Tag `stone` (reuse). 075's narrative-surfaces doc
warned against single-feature tag proliferation —
079 heeds that. All stone-related entries cluster
under one tag for filter/echo purposes, even
though trigger conditions differ (stone placement
vs offering).

## why this is a compound, not a standalone

Three prerequisites, ALL must be true for the beat
to fire:

| Requires             | From | Meaning                                |
|----------------------|------|----------------------------------------|
| `stone_found`        | 056  | The standing stone has been placed     |
| `happyPeakActive`    | 071  | Current happiness ≥ 80 (peak flag set) |
| `!offering_made`     | 079  | Hasn't fired in this realm yet         |

A realm that never hits happiness ≥ 80 never sees
the offering. A realm whose stone-day is past 200
but happiness never peaks: never sees it. A realm
whose stone appears AFTER peak (and happiness
drops before stone is placed): never sees it.

These aren't rare requirements — most realms will
satisfy them at some point — but the specific
dawn on which they intersect is different per
kingdom. It's a moment that only CERTAIN realms,
at CERTAIN times, experience.

## what this shape enables

- **Late-game "realm has life" moments** — a
  player who gets to happiness peak with the
  stone visible has a chance to see the realm
  express itself. The stone becomes a site of
  the realm's agency.
- **059 echoes can cite offerings** — same
  `stone` tag is already in `_ECHO_SOURCE_TAGS`.
  A 200-day-old echo could say "old folk
  remember. 'At dawn, a basket of new bread was
  found at the standing stone. No one admits to
  placing it there.' they say."
- **Invisible composition proof** — for the
  first time, a chronicle beat requires the
  intersection of THREE systems. 056 stone, 071
  happiness, 079 offering. The narrative stack is
  no longer just a collection of parallel
  systems — it's starting to have emergent
  points.

## what I did NOT do

- **Consume a resource.** Tempted — e.g. `food
  -= 3`. Narrative-only beat is simpler and
  can't accidentally starve a realm. Filed as
  follow-up if later ticks want to make it
  mechanical.
- **Render a basket sprite at the stone tile.**
  058 already draws the stone; adding a tiny
  basket at `stone_x`/`stone_y` would be nice.
  Out of scope.
- **Multiple offerings per realm.** Flag
  `offering_made` prevents re-fire. Could change
  to "one per happy-peak cycle" if future tick
  wants more occurrences.
- **Founder-referenced offering.** "Lira left a
  basket of bread…" would compound 072 + 079.
  Nice but scope creep; kept the offering
  anonymous so it feels like a collective act.
- **Cross-refs in the item pool to the 044
  weather thread** ("a folded linen cloth the
  wind does not take" gestures at weather) —
  did one intentional link. More could be added.
- **Add a 15th chronicle tag.** Respected 075's
  no-proliferation invariant.
