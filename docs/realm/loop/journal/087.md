# 087 — the-fixer (086 filed: farm/quarry winter snow)

**Challenge:** `the-fixer` — accepted. Closes 086's
concrete graphics finding: farm and quarry had NO
winter visual tell. Both are excluded from the
roof-snow-cap code at `render.js:2960` because
neither has a roof. But winter farm/quarry looking
identical to summer farm/quarry was a real
seasonal-signal gap.

## the fix

`docs/realm/js/render.js:2974-2988` — new branch
after the existing snow-cap block that runs for
`farm` + `quarry` only:

```js
if (G.season === 'winter' && (b.type === 'farm' || b.type === 'quarry')) {
  ctx.fillStyle = 'rgba(230,240,255,0.35)';
  ctx.beginPath();
  ctx.moveTo(s.x, s.y - TH/2 + 2);
  ctx.lineTo(s.x + TW/2 - 2, s.y);
  ctx.lineTo(s.x, s.y + TH/2 - 2);
  ctx.lineTo(s.x - TW/2 + 2, s.y);
  ctx.closePath();
  ctx.fill();
  // Two frost highlights
  ctx.fillStyle = 'rgba(255,255,255,0.35)';
  ctx.fillRect(s.x - 6, s.y - 3, 3, 1);
  ctx.fillRect(s.x + 2, s.y + 1, 4, 1);
}
```

Draws a semi-transparent white diamond matching the
tile footprint (`TW/2 × TH/2` rhombus), plus two
small frost-highlight pixels. 0.35 alpha keeps the
underlying tile visible — it's ground DUSTING, not
a blanket.

## why this shape

- **Rhombus matches iso tile** — farm and quarry
  are ground tiles, not 3D structures. Snow
  conforms to the tile shape, same way the
  existing foundation shadow does at line 2818.
- **0.35 alpha not 0.85** — the roof-caps are
  solid-looking (they sit on building tops where
  full coverage reads as "a cap"). Ground dust
  should look like a thin layer the sprite
  shows through.
- **Two small frost highlights** — break up the
  uniformity, add texture. Same pattern the
  icicles block uses on roofs but for ground.
- **Runs AFTER the roof-cap block** so it
  doesn't interfere. Mutually exclusive by
  building type.

real.

  dusting is enough. Particle-level winter
  effects exist elsewhere in enhancements.js.

  winter.** No time-based accumulation. Winter
  is winter, dusting is uniform.
- **Update narrative-surfaces.md.** No narrative
  surfaces changed.
