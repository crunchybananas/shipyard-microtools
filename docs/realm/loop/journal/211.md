## why this shape

- **Closes the longest-aged filing.** 060
  filed at tick 060; 211 ships 151 ticks later.
  Per 019's "long-waiting > recent" triage
  principle, this filing was overdue.
- **Sustained-state is a NEW BEAT SHAPE.**
  All 33 prior NARRATIVE_BEATS gate on
  POINT-IN-TIME conditions (`day >= X`,
  `season === Y`, `flag === true`). 211 gates
  on a DURATION (`day - lastRaidDay >= 50`).
  This requires runtime infrastructure
  (tracking when the previous event happened)
  not just state-snapshot. Future sustained-
  state beats can compose on the same shape.
- **Earned-peace gate.** `raidsSurvived >= 1`
  ensures the realm has FACED threat, not
  just lived in lucky absence of it. Without
  this, the beat would fire for any realm
  reaching day 50 — meaningless. With it, the
  beat marks a milestone of resilience.
- **`lastRaidDay !== undefined` defensive
  check.** Legacy saves (pre-211) don't have
  the field. Without the check, legacy saves
  with `raidsSurvived >= 1` would compute
  `G.day - undefined >= 50` = NaN >= 50 =
  false. Coincidentally safe by JS semantics
  but explicit-defense reads better.
- **Save persistence per 197 pattern.**
  Without persistence, reload would lose the
  peace-counter. Player accumulates 49 days
  of peace, saves, reloads, the beat never
  fires because `lastRaidDay` is gone. 197
  established the pattern; 211 follows.
- **Prose surprises.** "The realm carries the
  quiet differently than it carried the war"
  — peace as a verb, weight, burden. Per 203
  positive rule, the beat earns its landing.
  Compare to a template-completion phrasing
  ("Peace has held for fifty days") which
  would be patternic and rejected.
- **Misc tag joins observational-elder cluster.**
  The beat fits the cluster register:
  declarative, "the realm" subject, hopeful-
  resigned mood. Adds 8th misc-tag member;
  introduces 7th sub-type (sustained-state-
  recognition).
- **Realistic firing window: year-2+.** With
  raidInterval 8→4 days, raids happen ~every
  4-8 days. 50 days of peace requires extreme
  fortification (towers + barracks + walls)
  AND luck. The beat fires for STRONG late-
  game realms — different from year-1 cluster
  (196/199/207).
- **Doc invariant updated SAME tick** per
  204+208 process discipline. Cluster
  observed-pattern entry shows new sub-type
  + new use-count.
