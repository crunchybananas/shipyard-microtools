## why this shape

- **Silo body, not dome or sacks.** Silo
  is the granary's most-prominent visual
  element (large vertical mass). Dome is
  smaller (rooftop accent); sacks are
  decorative; grain piles are tiny. Silo
  varies most legibly.
- **4 colors per palette** matching the
  source 4-color gradient. Same shape as
  castle.roof (which also has 4-color
  source). Townhall.stone had 3-color
  source; the variant pool size matches
  source structure.
- **Faction-flavor names** mirror 279:
  warm-tan default / cool-grey northern /
  brick-red clay-fire / mossy-green
  forest. Future authors can lean into
  the regional-flavor framing if a beat
  references the realm's "grain
  storage tradition" or similar.
- **Silhouette preservation**: dome,
  sacks, grain piles, door, animation
  stay constant across variants. The
  granary stays a granary; only the
  CULTURAL TONE shifts.
