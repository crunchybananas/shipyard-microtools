## why this shape

- **Realm-naming-environment axis (third
  member).** Existing beats establish two
  naming axes:
  - 116 constellation_named: realm names a
    pattern in the sky. Year-1 autumn day ≥ 15.
  - 056 checkStoneBeat: realm finds a thing
    inscribed with the realm's own name. Day
    30-200 seeded.
  - **193 landmark_named: realm names a PLACE
    on the ground.** Year-2 day ≥ 50.
  Three naming events spanning sky → object →
  place. Each marks a different relationship
  between the realm and its surroundings.
- **Different mood from forgetting beats.** 184
  stone-forgotten + 190 constellation-forgotten
  + future founders-absence are all CLOSURE
  beats (something fades). 193 is OPENING
  (something comes-into-existence). The
  chronicle has been heavy with closure;
  variety welcome.
- **"No one decided this; it simply is" idiom.**
  Mirrors 148 wanderer ("no one calls them
  anything") + 152 night-shape ("no one asks
  them to describe it") in the elder-silence /
  emergent-knowledge register. Names accrete
  through use, not decision. Different from
  116 constellation's "${f}, looking up, names
  them..." which is a singular-act-by-named-
  character. 193 is collective-emergence-
  without-author.
- **8 rural-vernacular names.** Drawn from
  English folk-naming tradition (places named
  for distinguishing features: a bent ash, a
  cold spring, etc — not for founders, not for
  poetic invention). Kingdom-hashed pick via
  `_dreamHash(${kname}_landmark)` ensures
  determinism per kingdom.
- **Year-2 + day-50 gate.** Realm needs to have
  USED its territory long enough that places
  accrete colloquial names. Year-2 is day 29+;
  day ≥ 50 pushes well into year 2. Most long-
  lived realms see this fire.
- **Tag: milestone.** Matches 116's anchor.
  Fits the "realm crosses a recognition
  threshold" register.
