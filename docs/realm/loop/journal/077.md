## impact on a 200-day realm

Before 077, the chronicle got ~10-20 duplicate/wrong-
tag entries per realm from these sites:
- 2-5 duplicate raid/death death pairs (HIGH #1, #2)
- 4 pop-milestone duplicates at 10/25/50/75 (HIGH #3)
- Several "raid" entries for food/upgrade/trade/fire
  warnings (MEDIUM #4-8)

After 077, those ~10-20 entries disappear. Cleaner
chronicle for the same player experience. UI
toasts unchanged.

The net effect is that the realm's `tag:raid`
chronicle stream now actually contains RAID events
(not food warnings or trade failures). 059 echoes
become more coherent because when one says "the
bards still sing of that day: 'A rival lord sent
emissaries...'", the underlying beat is actually
about raids, not mis-categorized.
