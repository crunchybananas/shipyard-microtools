## the build

Three SVG edits.

### 1. `tower.svg` — lantern flicker

`<animate attributeName="opacity">` on the
lantern glow rect, with random-feeling key
times:
```svg
values="1; 0.92; 0.96; 1; 0.88; 0.95; 1"
keyTimes="0; 0.18; 0.32; 0.5; 0.68; 0.84; 1"
dur="1.4s"
```
Mostly steady (1.0) with occasional dimming —
reads as candle/oil-lamp behaviour, not
sine-pulse. 1.4s loop is short enough to be
seen but long enough to not flicker
distractingly.

### 2. `house.svg` — chimney smoke drift

3 wisps wrapped in `<g>` with paired
`<animateTransform translate>` + `<animate
opacity>`. 4.8s cycle drifting from (0,0) to
(-2, -10) px. Opacity values `[0, 0.85, 0.6, 0]`
fade in fast, hold, fade out. Staggered begin
times `0 / -1.6s / -3.2s` mean the 3 wisps
form a continuous plume rather than pulsing
together.

### 3. `bakery.svg` — chimney smoke drift

4 wisps (bigger plume than house — bakery oven
runs constant). 6s cycle, slightly slower than
house. Same translate + opacity pair, staggered
begin times `0 / -1.5s / -3s / -4.5s`. Larger
drift radius `(-3, -12)` to match the bigger
ellipses.
