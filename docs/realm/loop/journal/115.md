### 1. `docs/realm/js/audio.js` — new `founders` case

```js
case 'founders': {
  // C5, E♭5, G5 — rising minor triad
  makeOsc(ctx, dest, 'sine', 523, 0.06, 0.05, 0.9, t);
  makeOsc(ctx, dest, 'sine', 622, 0.06, 0.05, 0.9, t + 0.25);
  makeOsc(ctx, dest, 'sine', 784, 0.06, 0.05, 1.1, t + 0.50);
  break;
}
```

**Sound design — ceremonial-coded:**
- **Three notes** (523/622/784 Hz = C5/E♭5/G5) — one
  per founder slot. C-minor triad ascending. Solemn
  but not mournful.
- **0.25s spacing** — slower than stone's 0.08s quick-
  chime. Reads as "a ceremony unfolding" rather than
  "a thing happened."
- **0.05s soft attack** per note — each note eases in;
  no bell strike.
- **0.9-1.1s decay** per note — they overlap slightly
  creating a brief chord at the peak before fading.
- **Gain 0.06** — quietest cue yet (the founders are
  not proclaiming themselves loudly; they're introducing
  each other).
- **Pure sines.** Matches requiem and stone. Nightmare
  remains the only triangle-wave outlier.

## why this shape

- **One note per founder.** The 072 beat names THREE
  founders. Three notes is the obvious mapping. Adds a
  new acoustic STRUCTURE (multi-note sequence) to the
  contrast catalog — captured as a new row in the
  table.
- **Minor triad, not major.** The 072 prose is reverent
  but not triumphant ("the fire does not go out"). A
  minor triad carries gravitas without sadness; a major
  triad would feel too brass-band.
- **Ascending.** The three settlers are ARRIVING at
  each other. Descending would be departure; static
  would be a non-event. Up-motion = introduction.
- **Slow pacing.** 0.25s between notes is deliberate.
  Stone's 0.08s spacing is quick-chime; founders' 0.25s
  spacing is ceremony. Same underlying rhythm math,
  different affective register.
- **Matches `the-composer` challenge spec exactly** —
  114 said "respect audio-surfaces.md's 5 principles,
  update the acoustic-contrast table." All 5 principles
  honored (silent-by-default, miss-able at gain 0.06,
  once-per-realm via founders_named flag, distinct
  fingerprint as 1st multi-note cue, no UI), contrast
  table updated in same commit.
- **Pattern 1 + Pattern 1 + Pattern 1.** After 115,
  Pattern 1 count is 3 (nightmare, stone, founders).
  Pattern 2 still at 1 (requiem). Per 112's "3 is a
  pattern" rule, Pattern 1 now has enough usage to
  GENERALIZE — but Pattern 1 already IS the general
  case (late-bound import + try-catch). No refactor
  needed; the pattern scales naturally by repetition.
