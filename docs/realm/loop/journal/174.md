## why this shape

- **Sibling to 147 great-storm in seasonal
  placement.** Both fire year-2+ autumn. 174's
  `day >= 45` gate gives 147 a 2-day head start
  (147 fires day 43-44 typically; 174 fires day
  45+). Two beats land on different chronicle
  mornings within the same season.
- **Present-tense observation vs past-tense
  memory.** 147 is "Some of the realm remember a
  great storm…" — backward-looking. 174 is
  "There is a morning when frost finds the
  fields…" — present, immediate. They're a
  matched pair: memory + presence, the same
  realm-autumn from two angles.
- **Year-2-aware.** First frost gates on
  `year2` because the realm needs to have lived
  through one full cycle to RECOGNIZE the
  seasonal change. "The realm marks the year by
  it now" — the framing says THE REALM has
  developed a calendar. That's a year-2+
  awareness, not a year-1 raw experience.
- **Different family from 148/152/166.** Those
  are ambient-entity acknowledgment grammar
  (wanderer / ghost / frogs). 174 is seasonal/
  weather recognition. Diversifies the surprise
  palette without piling up the same pattern.
  156 warned against queue-drain; 174 honors
  the warning by going elsewhere.
- **Static string, no variants.** First frost
  is a singular event each year. Variants
  would imply multiple frosts; the FIRST frost
  is one moment per realm-year by definition.
  Matches 148/152's static-string restraint.
- **Tag: misc.** Same observational/communal
  register as 148/152/166. 075's reuse
  invariant honored.
- **No founder reference.** Continues the
  post-moratorium habit established at 157.
  Realm-as-collective is the subject; no
  named character.

realm's first reflective season — the one where

realms in autumn-of-year-1 see neither (only 116

signature).

(year-2+ via flag) + 093 second-winter beats +

diversifying meaningfully without crowding any
one focal axis.
