## why this surprises

This is the **FIRST LANGUAGE-DRIFT BEAT**
in the corpus. All previous habituation
beats observed:
- 254 ACTION domain (automaticity in
  routine — "fire is set without
  thinking")
- 280 TIME domain (rhythm-awareness in
  daily pauses — "no one has named this
  moment but everyone knows it")

285 introduces **LINGUISTIC domain**:
the realm's recognition that one of its
own phrases drifted from origin so far
that ORIGIN ITSELF became foreign. The
phrase is alive, in active use; the
ORIGINAL it diverged from is dead.

The lift line "would not know how to
use it now" captures a paradox: the
realm uses the phrase fluently but
could not USE the original phrase if
restored. Mishearing has displaced
hearing-correctly so completely that
correctness is no longer accessible.

This is distinct from FORGETTING (184
stone-forgotten / 190 constellation-
forgotten) where the THING is preserved
but KNOWLEDGE OF IT lapses. Here the
THING (phrase) is alive but its ORIGIN
has become inaccessible.
