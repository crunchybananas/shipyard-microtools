  agency ("the land has agreed to be lived

  bucket lowered to it, and remembers each one
  a long while"). Different shape within the
  same sub-type — the pattern is
  generalizable, not just the same prose
  twice. Promotes toward 3+ threshold.
- **Avoids 213 META-frame finding.** 212's
  "no one sees / the realm goes on without
  seeing" called out the chronicle's
  omniscience. 227 keeps the well's awareness
  WITHIN the diegesis — the well "remembers"
  but the chronicle doesn't editorialize about
  whether the realm knows.

  prose ("first thirsty summer") to land.
  Wells typically build day 5-15 in peaceful
  scenarios; year2 starts day 29; so the
  earliest fire is mid-year-2 spring. Most
  realms with wells will see this.
