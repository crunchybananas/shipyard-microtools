## why this shape

- **All-pushBox.** Skipped `pushFrustum` for now.
  The trapezoidal awning silhouette in the SVG
  (179) is a CSS/path effect that doesn't carry
  obvious benefit at iso-scale 3D — a flat box
  reads as "awning" against the empty-air-around-
  it that defines the open-air stall silhouette.
  pushFrustum stays filed; ship if/when a future
  building genuinely needs tapered cylinders or
  trapezoidal prisms.
- **Open-air silhouette preserved.** Counter
  (low) + 2 posts (vertical) + awning (raised
  slab) — the void between counter top and
  awning bottom is what reads as "open stall."
  Critical that posts are visually thin so the
  airspace dominates.
- **Awning ridge highlight as separate small
  box.** A bright `[0.95, 0.45, 0.30]` thin
  strip on top reads as "sunlit edge." Without
  emissive shading, this is the prototype's
  way of articulating roof apex.
- **Barrel prop.** Single small box, matches
  179 SVG's barrel + crate idiom. Could add
  crate too but kept to one accent prop for
  scope.
- **No coin pile / bunting / scallop fringe.**
  Those SVG-specific details (179) don't
  translate to per-vertex flat-color 3D.
  Acceptable abstraction loss.
