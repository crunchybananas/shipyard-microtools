# 224 — Phase C step 2 (castle pennants + tower banner sway)

## the build

Four `<animateTransform>` rotations, each
wrapping the cloth element (path + highlights)
of one pennant/flag/banner. Pole stays fixed;
cloth swings around the pole-tip pivot.

### `castle.svg`

- **Red pennant** (left tower) — pivot (47, 40),
  3.0s cycle, range -7° → +5°.
- **Blue pennant** (right tower) — pivot (81, 40),
  3.4s cycle, range -7° → +5°.
- **Gold flag** (central keep top) — pivot
  (64, 11), 4.0s cycle, range -5° → +4°
  (heavier cloth, slower + narrower swing).

### `tower.svg`

- **Red banner** — pivot (64, 56.5), 3.2s
  cycle, range -8° → +6°.

Asymmetric ranges (e.g., -7° to +5°) and
distinct durations across all 4 mean adjacent
buildings never sway in sync — reads as gusty
wind, not metronomic pulse.

### verify-anim.mjs upgrade

Parameterized for any sprite. Iterates through
windmill / castle / tower with their natural
animation cycle times. Asserts frame-diff via
PNG-byte-identical check.
