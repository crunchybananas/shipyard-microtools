## what bakery.svg expresses that canvas struggles with

- **Drawn bread-loaf sign vs emoji.** Canvas
  uses `ctx.fillText('🍞', s.x + 6, s.y - 16)`
  — depends on emoji-font rendering which is
  inconsistent across systems. SVG ships
  proper bread loaf with score lines + bracket
  + chain. Vector-clean at any zoom.
- **Bread paddle leaning against wall.**
  Canvas bakery has no exterior tools. SVG
  adds a long-handled bread paddle (flat
  blade ellipse + handle line) — iconic
  baker's implement.
- **Flour dust on doorstep.** Canvas has no
  flour. SVG adds 3 cream-white ellipses on
  the doorstep area implying "flour spills
  every day."
- **Half-timber posts on plaster walls.**
  Matches house/tavern/market family
  treatment. Canvas has flat plaster.
- **Domestic palette differentiation from
  blacksmith.** Both have chimney + glowing
  window. Bakery: terracotta + cream +
  yellow-warm + dough-smoke. Blacksmith:
  dark-stone + iron + amber-fire +
  coal-smoke. Same structural concept,
  opposite mood — domestic vs industrial.
- **Smoke that reads as flour-dust vs
  coal-soot.** Bakery smoke alpha 0.55
  starts white-grey; blacksmith smoke alpha
  0.6 starts dark-grey-brown. Same
  primitive (`<radialGradient>`) — different
  color rhetoric.
- **Warm-light-under-door** matches tavern
  175. Three buildings now share this
  "interior is OPEN" idiom: tavern (warm
  hearth glow + ajar door light), bakery
  (warm light spilling under door), market
  (open-air, no door at all but bunting +
  awning imply OPEN FOR BUSINESS).
