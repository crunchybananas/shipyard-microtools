**punctuation-pattern** keyTimes:

```svg

values="0 64 65.5; 0 64 65.5; -8 64 65.5;
```

bell is still. Then it swings 4 times with
decreasing amplitude (-8°/+6°/-5°/+3°) over

occasionally, not as constant motion.

arch. Arch + steeple stay fixed.
