## why this shape

- **One-number change.** The original 166 gate
  was `≥ 70`; the corrected gate is `≥ 35`. No
  other logic changes. Smallest possible fix.
- **Why 35, not some other number.** Year-2
  spring is days 29-35; day 35 qualifies as
  the last day of year-2 spring. Setting the
  gate to 35 means realms surviving to year 2
  spring/summer see the beat — a natural
  "settling year" framing. Lower (e.g., 25)
  would let the beat fire year-1 summer (days
  8-14, narrow window post-day-25); higher
  (e.g., 50) would push it into year 2 summer
  + year 3+. 35 is the cleanest threshold
  between "young realm" and "established."
- **Comment block expanded.** The comment now
  documents WHY the gate is 35, citing the
  original 70-mistake. Future readers see the
  reasoning so this doesn't get re-mis-set.
- **Doc updated same commit.** 075 invariant:
  narrative-surfaces.md updates in same commit
  as story.js NARRATIVE_BEATS changes.
  Compliance.
- **Cadence revised modestly.** +0.5-1 beat
  per realm is small. Total cadence still well
  under 300 cap.
- **No new beats.** Pure fixer; closes a
  filed bug, doesn't add content.
