## the build

Three SVG edits, one verify extension.

### 1. `tavern.svg` — sign-swing

Wraps the chain links + sign board + accent
band + outline + mug emblem in a parent
`<g>` with `<animateTransform type="rotate">`
±3° asymmetric around the chain-top
attachment (50, 89). 3.6s cycle. Different
period from castle/tower/market sway durations
so adjacent buildings don't sync.

The bracket (post + horizontal beam +
flourish triangle) stays OUTSIDE the swing
group — fixed structural support.

### 2. `bakery.svg` — flour-dust drift

The base-doorstep ellipse stays static (the
flour ON the doorstep). The two side patches
gain `<animate opacity>` (0.45 ↔ 0.7) +
`<animateTransform translate>` ±0.5px
horizontal drift, staggered begin times
(0/-2s) and varied dur (5.0s/5.4s) so the
patches drift asynchronously. Reads as flour
gently disturbed by foot-traffic + oven
draft.

### 3. `barracks.svg` — training-dummy idle sway

Wraps body + sack texture + head + target
marks + rope bindings + struck marks in a
parent `<g>` with `<animateTransform
type="rotate">` ±2.5° around (41, 87) — the
top of the dummy where it hangs from the
horizontal arm. 5.5s slow cycle reads as
passive hanging, not active swing. Post +
horizontal arm stay fixed (structural).

### 4. verify-anim.mjs

Added tavern + barracks. Now 9/9 sprites
verified.
