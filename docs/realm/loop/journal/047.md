## the fix

Two files, in the shape 045's journal prescribed.

`docs/realm/js/state.js` ‚Äî added `skyShift` triples
to each of four `SEASONS` entries:

```
spring: skyShift:[  0,  5,  5]   // barely, fresh
summer: skyShift:[ 10,  5, -8]   // warmer yellow cast
autumn: skyShift:[ 15, -5,-12]   // amber haze
winter: skyShift:[-10, -5, 15]   // cool blue
```

Magnitude ‚â§ 15/channel per 045 calibration. All
seasons still keep total channel magnitude well
within a "blue sky" read ‚Äî this is a tint, not a
re-skin.

`docs/realm/js/render.js:118-135` ‚Äî applied via
`shiftHex` to `dayTop`/`dayBot` only. dawn/dusk/night
constants are untouched:

```js
const seasonSkyShift = (getSeasonData().skyShift) || [0,0,0];
const shiftHex = (hex, sh) => {
  const r = Math.max(0, Math.min(255, ((hex >> 16) & 255) + sh[0]));
  const g = Math.max(0, Math.min(255, ((hex >> 8)  & 255) + sh[1]));
  const b = Math.max(0, Math.min(255, (hex & 255)         + sh[2]));
  return (r << 16) | (g << 8) | b;
};
const dayTopS = shiftHex(dayTop, seasonSkyShift);
const dayBotS = shiftHex(dayBot, seasonSkyShift);
// ...
apply(dayTopS, day); applyBot(dayBotS, day);  // only `day` blend uses shifted values
```

Why this shape ‚Äî the existing `apply()` lerps from
the current top/bot state toward a target color with
a blend coefficient. By passing only the SHIFTED
dayTop/dayBot to the `day` apply, the shift only
manifests when `day > 0` (t in [0.18, 0.82]). At
dawn/dusk peaks (t=0.22, t=0.78), the `day` blend
is partial, so shift bleeds through proportionally.
At dusk (t>0.82), `day=0` ‚Üí no leak. At midnight,
`day=0` ‚Üí no leak.

This is exactly the behavior 045 specified: seasons
carry at midday, disappear at pure dusk/night, bleed
subtly at dawn where the day and dawn bands overlap.

## why these magnitudes

| Season | R   | G   | B    | Read                            |
|--------|-----|-----|------|---------------------------------|
| spring | +0  | +5  | +5   | Slight fresh/pale ‚Äî subtle      |
| summer | +10 | +5  | -8   | Warm yellow, less blue          |
| autumn | +15 | -5  | -12  | Amber haze (biggest R lift)     |
| winter | -10 | -5  | +15  | Cool, more saturated blue       |

Chose asymmetric magnitudes so autumn pushes WARM
(max R lift, B cut) and winter pushes COOL (max B
lift, R cut) ‚Äî those are the seasons players most
need to identify at a glance. Spring and summer sit
in the middle band with smaller shifts, matching
their perceptual similarity in nature.
